📖 Guía de Instalación y Configuración

🌐 ¿Cómo funciona este sistema?

Este es un archivo HTML único que funciona como una app web. Los datos se guardan en Firebase Firestore (nube de Google). Los técnicos lo usan desde su celular y los administradores ven todo en el panel.

1

Crear proyecto Firebase

Ve a console.firebase.google.com → Crear proyecto → Nombre: ecosem-ot

💡 Usa tu cuenta Google de ECOSEM para que sea de la empresa.
2

Obtener credenciales

En Firebase: Configuración del proyecto → Tu app web → Registrar app → Copia el apiKey y projectId

apiKey: "AIzaSy..." projectId: "ecosem-ot"
3

Activar Firestore

En Firebase: Firestore Database → Crear base de datos → Modo de producción → Región: us-central1

⚠️ Luego ve a Reglas y cambia allow read, write: if false por allow read, write: if true (temporal para pruebas)
4

Configurar en la app

En esta app → tab Config → Pega el API Key y Project ID → Define la clave admin → Guardar

✅ Los datos se guardan en este dispositivo. Cada celular necesita hacer esto una sola vez.
5

Publicar en Cloudflare Pages

Sube este archivo como index.html a GitHub → Conecta tu repo en Cloudflare Pages → Se despliega automáticamente

github.com → marchuaman15-web/ecosem-huaraucaca-v6 → Reemplazar index.html → Push → Deploy
🌐 Todos los técnicos acceden desde su celular con la URL de Cloudflare.
6

Distribuir a técnicos

Comparte la URL por WhatsApp. Los técnicos pueden agregar la página a su pantalla de inicio como si fuera una app.

📱 En Chrome: menú (...) → Agregar a pantalla de inicio → Icono de ECOSEM en el celular
7

Módulos disponibles

El sistema tiene 5 módulos independientes:

✅ CHECKLIST — Inspección B/M/NA por equipo 🔧 MANTENIMIENTO — OTs, historial, costos 🔵 LLANTAS — Control por posición y vida útil 📦 REPUESTOS — Stock, alertas, movimientos 📊 DASHBOARD — KPIs, gráficos, informes PPT
